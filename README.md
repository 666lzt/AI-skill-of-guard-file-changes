# AI-skill-of-guard-file-changes
This is a cooperative safety skill for AI-assisted development.
This skill is used for file security protection in AI Agent project development. It records file creation, modification, deletion, and movement operations, performs snapshots and permission checks before deletion, supports rollback based on snapshots, and retains audit records using JSONL logs. It is suitable for preventing AI from accidentally deleting or modifying project files or files outside the sandbox.
